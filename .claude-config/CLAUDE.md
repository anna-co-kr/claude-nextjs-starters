## 개발 환경
- OS: macOS
- 언어: TypeScript
- 프레임워크: Next.js 15, React 19

## 언어 및 커뮤니케이션 규칙
- 기본 응답 언어: 한국어
- 코드 주석: 한국어로 작성
- 커밋 메시지: 한국어로 작성
- 문서화: 한국어로 작성 (IMPORTANT)
- 변수명/함수명: 영어 (코드 표준 준수)

## 코딩 스타일
- 들여쓰기: 2칸
- 네이밍: camelCase, PascalCase (컴포넌트)
- 선호 프레임워크: React, Next.js

## 기술 스택
- CSS: Tailwind CSS
- UI: shadcn/ui
- 상태관리: Zustand
- 폼: React Hook Form + Zod

## 추가 사항
- any 타입 사용 금지
- 컴포넌트 분리 및 재사용
- 반응형 필수
